# Binary Search Tree

A JavaScript implementation of a Binary Search Tree (BST) completed as part of The Odin Project.

The project demonstrates how a binary search tree works internally, including tree construction, insertion, deletion, searching, traversals, balancing and recursive algorithms.

## Features

- Build a balanced binary search tree from an array
- Automatically remove duplicate values
- Automatically sort the input array
- Search for values
- Insert new values
- Delete existing values
- Calculate node height
- Calculate node depth
- Check whether the tree is balanced
- Rebalance an unbalanced tree
- Tree traversals:
    - Level-order
    - In-order
    - Pre-order
    - Post-order

## Concepts Practiced

- Binary Search Trees
- Recursion
- Tree traversal algorithms
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Recursive problem solving
- Object-oriented programming
- Private class methods
- Data structures and algorithms

## Available Methods

- `buildTree(array)`
- `includes(value)`
- `insert(value)`
- `deleteItem(value)`
- `levelOrderForEach(callback)`
- `inOrderForEach(callback)`
- `preOrderForEach(callback)`
- `postOrderForEach(callback)`
- `height(value)`
- `depth(value)`
- `isBalanced()`
- `rebalance()`
